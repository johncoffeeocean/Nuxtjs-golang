# nuxt-golang-sample

A sample application for Typescript(Nuxt.js), Golang(gin, gorm) and MySQL5.7 runs in docker.

## Included

- MySQL 5.7
- Golang 1.16.0
- Node 15.11.0

## Running the app in local development

- build

```sh
$ docker-compose build
```

- run app

Simply run:

```sh
$ docker-compose run
```

By default you can access server via http://localhost:3000
